Welcome to Molly Project's documentation!
=========================================

The Molly Project is an open source batteries included framework for the
rapid development and deployment of mobile and desktop portals originally
developed by the University of Oxford. It was originally developed for
Higher Education institutions but is suitable in many other situations.
